# المرجع الرقابي الذكي — نسخة MVP

تطبيق عربي خفيف قابل للتشغيل محليًا أو على Replit دون الحاجة إلى Replit Agent.

## الوظائف الحالية

- بحث عربي في 52 مصدرًا رسميًا للهيئة، تشمل موقعها وحسابها الرسمي على منصة X.
- بحث داخل 40 مقتطفًا نظاميًا موثقًا يغطي المجالات السبعة.
- محرك فئوي لأسماء المنتجات الشائعة في المجالات السبعة، مع تصحيح إملائي ومسارات رسمية موجهة ونتيجة آمنة للمنتج غير المعروف.
- معالجة المصطلحات المتداخلة مثل «محاليل / IV» والتمييز بين المحلول الدوائي وطقم أو مضخة التسريب الطبية.
- مكتبة بحث داخلية تضم 138 قاعدة وأمثلة مستخلصة من دليل تصنيف المنتجات الرسمي، مع القسم والصفحة والرابط الرسمي.
- ملف رقابي شامل لكل منتج يقسم الارتباطات إلى التصنيف، والتسجيل والفسح، والأنظمة واللوائح، والأدلة والاشتراطات، والمواصفات والمكونات، والتحذير والسحب.
- بحث حي داخل التطبيق يعرض السجلات الرسمية الفعلية من التحذيرات والسحب والاستدعاء وتحذيرات السلامة الدوائية والأنظمة واللوائح والأدلة والتعاميم، مع تنبيه يمنع اعتبار عدم ظهور نتيجة إثباتًا لعدم وجود سحب أو تعميم.
- إنشاء محضر تحقق أولي لمنتج يتضمن الشركة والتشغيلة ووقت البحث وقائمة المصادر الرسمية التي فحصها المفتش.
- تصفية حسب المجالات الرقابية الثمانية وفئة المصدر.
- ترتيب النتائج حسب الصلة ومستوى الحجية.
- تصنيف استرشادي للمنتج يغطي الغذاء والأعلاف والمبيدات والأجهزة الطبية والتجميل والمستحضرات الصيدلانية والبيطرية.
- مسار تحقق موجّه يفتح قواعد التسجيل والسحب والتحذيرات المناسبة للمجال.
- مساعد تحقق بصري لرفع أو تصوير المنتج وقارئات الحرارة ومناطق الاستلام والتخزين والعزل والثلاجات، مع ربط الملاحظات بالمتطلبات الرسمية وطلب الأدلة الإضافية.
- ربط كل مصدر بمحاور بنك أسئلة المفتشين.

## التشغيل

```bash
npm start
```

ثم افتح المنفذ `3000`.

## الاستيراد إلى Replit

1. ارفع ملفات المشروع إلى مستودع GitHub خاص.
2. في Replit اختر **Create App** ثم **Import from GitHub**.
3. اختر المستودع وشغّل الأمر `npm start`.

## حدود النسخة الحالية

- سجل المصادر محصور في روابط عامة رسمية على نطاق SFDA.
- فهرس النصوص الحالي جزئي، ولا يغطي النص الكامل لجميع المرفقات أو جميع المجالات بعد.
- البحث التنظيمي يقرأ السجلات المنشورة في قواعد الهيئة المدعومة؛ أما حالة تسجيل المنتج فتظل مسار تحقق مستقلًا بحسب قاعدة المجال.
- نتيجة التصنيف استرشادية فقط ولا تمثل قرارًا رسميًا.
- نتيجة التحليل البصري قرينة مساندة فقط، ولا تمثل إثبات مطابقة أو مخالفة دون تحقق المفتش.
- لا تستخدم النتيجة وحدها لاتخاذ إجراء نظامي أو توثيق مخالفة.

## الخطوة التالية

توسيع فهرسة النصوص لبقية المجالات والمرفقات الرسمية، ثم لوحة إدارة ومراجعة دورية للمصادر، وبعدها الربط المعتمد بقواعد المنتجات المتاحة.
